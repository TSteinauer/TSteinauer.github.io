Bad Website Review -

Yale School of Art

This website serves as a hub of resources for students interested in joining the Yale school of art

  • CONTRAST: This website does have colors that allow the text to stand out from the backgroud, however there are so many different colors and contrast that it becomes distracting
  • REPITITION: There is little to no consistency in this website and has so many colors and fonts
  • ALIGNMENT: There is no visual connection in this web page and is skewed more to the left. A lot of visual vomit here
  • PROXIMITY: While spacing is present to attempt to seperate different content, a lot of similar content that should be grouped together tends to be spread apart from each other such as the zoom links

Bad Website Review -

Virgin Airlines

This website serves as a page for customers to access all content possible about Virgin Airlines

  • CONTRAST: This website has great contrast and is consistent with black and white with a tasteful amount of red that really makes the pages stand out while also be readable
  • REPITITION: The fonts, colors, and even the graphics are very consistent and there is a visible theme to the site. Overall the design remains consistent throughout.
  • ALIGNMENT: The sight is very symmetrical and contains a centered allignment between all it's elements.
  • PROXIMITY: All the contents of the website are grouped very nicely and allow for users to click on single squares to access different content which makes the sight extremely clean and organized
